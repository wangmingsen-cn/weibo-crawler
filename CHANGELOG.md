# Changelog

所有版本更新记录。

## [1.0.0] - 2024-04-20

### ✨ 新增功能
- 支持微博单条内容爬取
- 支持微博用户主页批量爬取
- 支持头条文章爬取
- 支持头条视频爬取
- 智能URL识别，自动判断平台
- JSON和TXT双格式输出
- 命令行和Python API两种使用方式

### 🛡️ 技术特性
- 随机延时防封策略
- 请求失败自动重试
- 多种数据提取方式（提高成功率）
- HTML标签自动清理
- 转发微博原内容保留

### 📦 项目结构
- `crawler.py` - 主程序（单文件，无需安装）
- `README.md` - 详细使用文档
- `LICENSE` - MIT开源协议
- `requirements.txt` - 依赖列表
- `examples/` - 使用示例
